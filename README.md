# Домашнее задание №09 - Разложить число на простые множители

Прочитать с клавиатуры число `N`. Вывести через пробел по возрастанию все простые делители числа
`N` столько раз, сколько они входят в его разложение на простые множители. После последнего числа
символа пробела быть не должно.

Вывод должен заканчиваться переводом строки.

## Пример

### Входные данные

```
60
```

### Выходные данные

```
2 2 3 5

```
